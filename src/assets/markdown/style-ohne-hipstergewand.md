# Style ohne Hipstergewand

Wow, was für ein biestiger Sound. =D

Die Engländer von CRIM3S treten mit Lo-Fi Electro direkt in die Fußstapfen von Crystal Castles, nur mit einer total dreckigen Raveattitüde, die richtig Spaß macht! Völlig konträr zu irgendwelchen Mainstream Pop-Grazien untermalt die Sängerin die Musik völlig ungehemmt mit Shouts und Gesangspassagen. Dabei nehmen sie sich überhaupt nicht wichtig, sondern machen einfach das worauf sie Bock haben, Respekt!

Den Anfang macht DOSE: 90er Jahre Trance Sirene und abgeschnittener Bass wie aus einem kaputten Ghettoblaster, fettest!

Ähnlich massiv ist still going. Langsam angefangen baut sich ein wahres trapähnliches Monster auf, das jedoch nicht an Eleganz einbüßt.

Das Konzept bleibt erhalten und es wird nebenbei noch eine Message mit Stay Ugly rausgehaun. End geil trashig.
